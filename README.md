ZEPTO_ANALYSIS
---------------------
This project analyzes Zepto product and sales data using PostgreSQL.
The goal was to perform SQL-based data cleaning and extract business insights from the dataset.

Data Workflow
---------------------
1.Downloaded dataset from Kaggle

2.Cleaned and verified structure in Excel

3.Imported dataset into PostgreSQL using pgAdmin

4.Performed SQL queries for data cleaning and analysis

5.Extracted business insights

TOOLS AND TECHNOLOGIES USED
----------------------------
* PostgreSQL

* pgAdmin 4

* SQL

* Microsoft Excel (for dataset format handling)

* Kaggle (dataset source)

* GitHub
